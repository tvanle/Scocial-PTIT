## 1. GIỚI THIỆU TỔNG QUAN

### 1.1 Module Dating là gì?

Dating Module là một tính năng **hẹn hò trực tuyến** được tích hợp trong ứng dụng PTIT Social, cho phép sinh viên PTIT tìm kiếm và kết nối với nhau.

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│                    PTIT SOCIAL APP                              │
│                                                                 │
│   ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────────────┐   │
│   │  Feed   │  │  Chat   │  │ Profile │  │     DATING      │   │
│   │ (Bảng  │  │(Nhắn tin│  │(Hồ sơ  │  │   (Hẹn hò)      │   │
│   │  tin)   │  │  nhóm)  │  │cá nhân) │  │                 │   │
│   └─────────┘  └─────────┘  └─────────┘  └─────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Đối tượng người dùng

| Đối tượng | Mô tả |
|-----------|-------|
| **Sinh viên PTIT** | Tất cả sinh viên đang học tại PTIT |
| **Độ tuổi** | 18 - 30 tuổi |
| **Mục đích** | Tìm bạn bè, người yêu, networking |


## 2. VẤN ĐỀ VÀ GIẢI PHÁP

### 2.1 Các vấn đề cần giải quyết

```
┌─────────────────────────────────────────────────────────────────┐
│                       VẤN ĐỀ                                    │
└─────────────────────────────────────────────────────────────────┘
        │
        ├─── 1. Làm sao để tìm được người phù hợp?
        │         → Thuật toán SCORING với 6 yếu tố
        │
        ├─── 2. Làm sao biết ai thích mình?
        │         → Hệ thống MATCHING 2 chiều
        │
        ├─── 3. Làm sao tránh người lạ (không phải SV PTIT)?
        │         → Xác thực EMAIL @ptit.edu.vn
        │
        ├─── 4. Làm sao để trải nghiệm mượt mà như Tinder?
        │         → Animation SWIPE với physics engine
        │
        └─── 5. Làm sao thanh toán Premium dễ dàng?
                  → Tích hợp VNPay + VietQR
```

### 2.2 Giải pháp đề xuất

| Vấn đề | Giải pháp | Công nghệ |
|--------|-----------|-----------|
| Tìm người phù hợp | Thuật toán scoring 6 yếu tố | Multi-factor algorithm |
| Biết ai thích mình | Mutual like = Match | Database matching |
| An toàn | Chỉ SV PTIT (@ptit.edu.vn) | Email verification |
| Trải nghiệm mượt | Swipe cards như Tinder | React Native Reanimated |
| Thanh toán | VNPay + VietQR | Payment gateway integration |
